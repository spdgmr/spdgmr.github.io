Projects

Here are a few projects I've worked on along with a short description.

(free means free as in freedom, non-free means I cannot meet all requirements necessary to consider it free software)

  • Project 081

    • Modification of Mac OS X 10.4 Tiger which runs on Early 2008 Apple Macs. Non-free although my code is free. (non-free)

  • elevendebloater

    • Microsoft Windows 10/11 debloater tool which cleans up spyware, tracking and bloatware that Windows comes with. (free, runs on proprietary OS)

  • rchat

    • Minimal and simple rautafarmi client written in Bash that follows the suckless philosophy. (free)

  • MagmaBoar

    • "MagmaBoar is a privacy and freedom respecting Mozilla Firefox profile that completely removes all non-free software through the use of LibreJS. (free)

  • spDE

    • "speedie's Desktop Environment" or "spDE" is a set of POSIX shell scripts to automatically install my dotfiles on Gentoo, Arch, Void, Debian or Red Hat. (free)

  • xshbar

    • Customizable, minimal and modular xsetroot bar designed to be used with window managers that use xsetroot -name for displaying information. (free)
